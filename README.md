# Secure Login System

## Features

- User Registration
- User Login
- Password Hashing using bcrypt
- SQLite Database
- SQL Injection Protection using Parameterized Queries
- Session Management
- Logout Feature

## Installation

```bash
pip install -r requirements.txt
```

Run the database:

```bash
python database.py
```

Run the application:

```bash
python app.py
```

Open:

```
http://127.0.0.1:5000
```
